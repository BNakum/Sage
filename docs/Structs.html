  • Bitboard

    A bitmap of sixty-four bits suitable for storing squares for various pieces.

    The first bit refers to Square.A1 the last (64th) bit refers to Square.H8.

    Due to their compact nature, bitboards can store information such as positions in memory very efficiently. Bitboards can also be used to answer questions about the state of a Board quickly with very few operations.

    Bitboards used internally within Board to store positions for all twelve cases of Piece.

    See also

    Bitboard (Wikipedia), Bitboards (Chess Programming Wiki)
